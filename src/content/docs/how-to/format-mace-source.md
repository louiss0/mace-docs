---
title: How to Format Mace Source
description: Print canonical formatted Mace source from a file.
---

Use the `source` command when you want normalized Mace formatting.

## Format a file

```bash
mace source ./hello.mace
```

## When to use it

Use this when you want to:

- normalize whitespace and layout
- inspect the canonical source form
- compare formatting changes consistently
