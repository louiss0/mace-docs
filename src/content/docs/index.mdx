Mace is a typed configuration language for deterministic object data.
These docs are written from the language contract in `mace-spec.md` and the
current CLI behavior.

## What Mace includes

- A typed language with imports, script declarations, and one output block.
- A `mace` CLI for JSON output, importing source formats, AST inspection,
  source normalization, and language server runtime.
- A downloadable Language Server Protocol (LSP) experience through the Mace CLI.
- Official Node and Python bindings in `bindings/node` and `bindings/python`.

## LSP, CLI, and bindings at a glance

The Mace LSP is part of the CLI. Download/install the CLI, then run:

```bash
mace lsp
```

If you prefer language bindings, use:

- Node bindings: `@code-fixer-23/mace-node`
- Python bindings: `mace-python`
